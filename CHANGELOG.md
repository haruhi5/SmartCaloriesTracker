## [1.0.0] - 2024-01-21

### Added
- Initial release of Smart Calories Tracker
- AI-powered food recognition using GPT-4 Vision and Gemini
- Camera and gallery image capture for food analysis
- Daily calorie tracking with progress visualization
- TDEE/BMR calculator based on user profile
- User profile management (height, weight, age, gender, activity level)
- Fitness goal settings (maintain, muscle gain, fat loss)
- CSV data export functionality
- Material 3 / Material You theming with dynamic colors
- Room database for local data storage
- Hilt dependency injection
